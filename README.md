# Yessin WP Local Stack

This repository contains a Docker-based local WordPress environment tailored for the custom `yessin-starter-theme`. It includes:

- `docker-compose.yml` to boot a WordPress + MySQL stack with HTTPS via mkcert certificates.
- `docker/wp` custom image that enables Apache SSL and autoloads the theme.
- `wp-content/themes/yessin-starter-theme` – the bespoke WordPress theme with sticky navigation and custom blocks.

## Prerequisites

- Docker Desktop 4.x+
- Node.js 18+ (for theme asset builds)
- Composer 2+
- [mkcert](https://github.com/FiloSottile/mkcert) for local certificates (optional if you supply your own)

## First Run

```bash
# bring up the stack
docker compose up -d --build

# create WordPress tables & install theme assets
docker compose exec wp wp plugin deactivate --all
```

Visit `https://leidenamnestynl.test` (map `127.0.0.1 leidenamnestynl.test` — SwitchHosts! helps automate this).

## Theme Development

```bash
cd wp-content/themes/yessin-starter-theme
npm install
npm run build
```

To work on PHP / JS / CSS, edit within the theme directory. The sticky menu JS lives in `assets/js/main.js`, styles in `style.css`.

## Environment Variables

WordPress and MySQL credentials are defined in `docker-compose.yml`. Adjust as needed and restart the stack.

## Optional: mkcert Certificates

Certificates generated by mkcert live in `certs/` (ignored by git). If you clone this repo elsewhere, run:

```bash
mkcert -install
mkcert leidenamnestynl.test
mv leidenamnestynl.test*.pem certs/
```

Then restart the containers to reload TLS.
